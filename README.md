# mkotto.github.io

A personal landing page and digital visit card featuring an interactive animated background and links to social media profiles.

## Live Site
Visit the page at: [https://mkotto.github.io/](https://mkotto.github.io/)

## Features

- **Animated Gradient Background**: Rotating gradient animation with smooth color transitions
- **Interactive Mouse Tracking**: Glowing radial gradient effect that follows cursor movement with breathing animation
- **Social Media Links**: Quick access to LinkedIn, Last.fm, GitHub, and Steam profiles
- **Responsive Design**: Built with Bootstrap 5.3.3 for mobile-friendly display
- **Clean & Minimal**: Simple, focused design that puts content first

## Technologies

- HTML5
- CSS3 (with keyframe animations)
- Vanilla JavaScript
- Bootstrap 5.3.3

## Local Development

Simply open `index.html` in your browser to view the page locally.

## Deployment

This site is deployed using GitHub Pages. Any changes pushed to the main branch are automatically published.
